Deploy ZeroClaw from the attached scaffold pack.

1. Extract the zip to /a0/usr/workdir/zeroclaw/
2. Create a Python venv at /a0/usr/workdir/zeroclaw/venv and install deps from requirements.txt
3. Run: bash run_zeroclaw.sh
4. Verify both ZeroClaw (port 9000) and OpenClaw (port 9100) are responding
5. Open http://localhost:9000/dashboard and confirm it loads
