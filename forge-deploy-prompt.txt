Deploy ZeroClaw from the attached scaffold pack.

1. Place the zip in /a0/usr/uploads/ (if not already there)
2. Run: python start.py
   - Auto-discovers the newest zip in /a0/usr/uploads/
   - Extracts it to the project directory
   - Starts ZeroClaw (port 9000) and OpenClaw (port 9100)
3. If deps are missing: pip install -r requirements.txt
4. Verify: python start.py --check
5. Open http://localhost:9000/dashboard and confirm it loads

Alternative (explicit zip path):
   python start.py --zip /a0/usr/uploads/dashboard_scaffold.zip
